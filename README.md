### Simple Graduation Countdown App

## Prerequisites

- [Node.js](https://nodejs.org/en/)

## Installation

clone the repo

```bash
git clone https://github.com/MohAlkurdi/graduation-countdown-rn.git
```

cd into the project

```bash
cd graduation-countdown-rn
```

install the dependencies

```bash
npm i
```

## Usage

```bash
npm start
```
